Sorteador de números

Projeto criado nas aulas do Devclub. Uma aplicação web simples que gera números aleatórios dentro de um intervalo definido pelo usuário. O JavaScript é responsável pela lógica do sorteio, capturando os valores inseridos, gerando o número aleatório e exibindo o resultado na tela de forma dinâmica. O projeto foi criado para praticar lógica de programação, manipulação do DOM e interação com o usuário.
